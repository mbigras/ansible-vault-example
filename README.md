# Ansible Vault Example

> Hacking on ansible-vault

## Usage example

```
ansible-playbook --connection local playbook.yml

PLAY [all] **************************************************************************************************************************************************************************

TASK [Print something] **************************************************************************************************************************************************************
ok: [localhost] => {
    "msg": "hello cats"
}

TASK [Run an encrypted executable] **************************************************************************************************************************************************
changed: [localhost]

TASK [Verify executable output] *****************************************************************************************************************************************************
ok: [localhost] => {
    "msg": "hello world\n"
}

PLAY RECAP **************************************************************************************************************************************************************************
localhost                  : ok=3    changed=1    unreachable=0    failed=0

```
